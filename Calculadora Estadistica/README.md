<h1>Calculadora estadistica</h1>

<p>Es este codigo podemos ver el uso de apuntadores y vectores para la creacion de una calculadora estadistica la cual podra calcular lo siguiente: </p>

<ul>
  <li>Media</li>
  <li>Variancia</li>
  <li>Desviacion</li>
</ul>

<p>Tambien tiene una funcion de lectura y una de imresion del vector donde se almacenan todos los datos ingresados</p>
